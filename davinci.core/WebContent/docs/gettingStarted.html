Getting Started

  1. Read Introduction and Overview.
  2. Read Accessing/Installing the build to determine how you will run deVinci - either from your local system, or from the Maqetta server. To get up and running quickly, chose running from the server.
  3. Launch Maqetta from the server, or download and install Maqetta and launch it locally.
  4. Read The Maqetta Workspace to learn about the Maqetta user interface.
  5. Go through the Page Editor Overview tutorial to get a quick tour of Maqetta.
  6. Go through the Mocking Up a Task Manager and Adding Interactivity tutorials to learn how to mock up an interactive page from start to finish.
  7. Read The Maqetta Theme Editor to learn about using Maqetta themes.
  8. Go through the Theme Editor Tutorial.
