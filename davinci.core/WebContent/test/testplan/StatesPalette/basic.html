States palette basic tests

Creating states, making widgets invisible within a state

Steps

  1. Standard setup things: Clear the browser cache. Launch the daVinci server. Login if necessary. In some cases, you'll want to setup a virgin user environment by creating a whole new user login.
  2. Issue a "New HTML File" command.
  3. Drag an HTML label, Calendar, TextBox, a DIV, and three buttons inside of DIV onto the canvas.
  4. In the States palette, click on the + icon to define two states "aaa" and "bbb"
  5. In the States palette, select the "bbb" state and click on the - icon to delete that state
  6. In the States palette, click on the + icon to define two states "ccc" and "ddd"
  7. In the States palette, select the "ccc" state and click on the - icon to delete that state.
  8. In the States palette, select the "aaa" state. Go to Outline palette and click on the eyeball icon for the Label to make it invisible for the "aaa" state. (The Label should become invisible on the canvas).
  9. In the States palette, select the "ddd" state. Go to Outline palette and click on the eyeball icon for the Calendar to make it invisible for the "ddd" state. (The Calendar should become invisible on the canvas).
  10. Click on the "Save" icon on the main toolbar to save the new file. The asterisks that indicates that the file is "dirty" should go away.
  11. Click on the "X" close button for the editor tab for this file
  12. In the Files palette, double-click on the file you just created to re-open the file.
  13. In the States palette, alternatively click on "Normal", "aaa" and "ddd".

Expected results

  1. The TextBox and buttons should be visible in all 3 states. The Label should disappear in the "aaa" state. The Calendar should disappear in the "ddd" state.

(continuation of previous) Adding interactivity

Steps

  1. (continuing on the same file created in the previous test)
  2. Select the first button widget. On the Properties palette, go to the onclick property and use the dropdown menu to choose the option to switch to the "Normal" state
  3. Select the second button widget. On the Properties palette, go to the onclick property and use the dropdown menu to choose the option to switch to the "aaa" state
  4. Select the third button widget. On the Properties palette, go to the onclick property and use the dropdown menu to choose the option to switch to the "ddd" state
  5. Save and close the file.
  6. In the Files palette, double-click on the file you just closed to re-open the file.
  7. Click on the Preview in Browser button on the main toolbar. A new browser tab or window should appear with your new file showing within the frame.
  8. Alternatively click on the 3 buttons.

Expected results

  1. The TextBox and buttons should be visible when the page launches and after clicking the first button. The Label should disappear when you click on the second button. The Calendar should disappear when you click on the third button.

Changing properties in different states

Steps

  1. (continuing on the same file created in the previous test)
  2. In the States palette, select the "ddd" state. Go to Outline palette and click on the closed eyeball icon for the Calendar to make it visible (the eyeball should open and the Calendar should become visible on the canvas), then click on the open eyeball icon for the TextBox to make it invisible (the eyeball should close and the TextBox should become invisible on the canvas).
  3. (Note: For some of these cases below, the Properties palette will default to targeting the CSS style rule within a CSS style sheet. Instead, bring up CSS cascade and set the target to element.style. This might be something we fix in the future. We should know that you can set fonts on the root node for button and can set width on the root for calendar.)
  4. On the States palette, go into the Normal state. Select the first button widget. On the Styling palette, set font color to a reddish color and font size to 20. Select the calendar widget. On the Properties palette, set the width to 200px.
  5. On the States palette, go into the "aaa" state.
  6. Select the second button widget.
  7. In Properties palette, open up the CSS Cascade view for the 'color' property by clicking on ">" symbol to right of 'color' property. Make sure radio button to left of 'element.style' is selected. Scroll to very bottom of CSS cascade and turn on check box that says 'Only apply to current style (aaa)'. Set font color to a greenish color.
  8. Open up the CSS Cascade view for the 'font-size' property by clicking on ">" symbol to right of 'font-size' property. Make sure radio button to left of 'element.style' is selected. Scroll to very bottom of CSS cascade and turn on check box that says 'Only apply to current style (aaa)'. Set font size to 30.
  9. Select the calendar widget.
  10. Open up the CSS Cascade view for the 'width' property by clicking on ">" symbol to right of 'width' property. Make sure radio button to left of 'element.style' is selected. Scroll to very bottom of CSS cascade and turn on check box that says 'Only apply to current style (aaa)'. Set the width to 300px.
  11. On the States palette, go into the "ddd" state.
  12. Select the third button widget.
  13. Open up the CSS Cascade view for the 'color' property by clicking on ">" symbol to right of 'color' property. Make sure radio button to left of 'element.style' is selected. Scroll to very bottom of CSS cascade and turn on check box that says 'Only apply to current style (ddd)'. Set font color to a bluish color.
  14. Open up the CSS Cascade view for the 'font-size' property by clicking on ">" symbol to right of 'font-size' property. Make sure radio button to left of 'element.style' is selected. Scroll to very bottom of CSS cascade and turn on check box that says 'Only apply to current style (ddd)'. Set font size to 40px.
  15. Select the calendar widget.
  16. Open up the CSS Cascade view for the 'width' property by clicking on ">" symbol to right of 'width' property. Make sure radio button to left of 'element.style' is selected. Scroll to very bottom of CSS cascade and turn on check box that says 'Only apply to current style (aaa)'. Set the width to 400px.
  17. Save and close the file.
  18. In the Files palette, double-click on the file you just closed to re-open the file.
  19. Click on the Preview in Browser button on the main toolbar. A new browser tab or window should appear with your new file showing within the frame.
  20. Alternatively click on the 3 buttons.

Expected results

  1. You should see slightly different results than in the previous section. The Label should disappear when you click on the second button and the TextBox should disappear when you click on the third button.
  2. In all 3 states, the first button should have a reddish color and a font size of 20
  3. In the normal state, the Calendar whould be 200px wide.
  4. In the "aaa" state, the second button should have a greenish color and a font size of 30 and the Calendar whould be 300px wide.
  5. In the "ddd" state, the third button should have a bluish color and a font size of 40 and the Calendar whould be 400px wide.
