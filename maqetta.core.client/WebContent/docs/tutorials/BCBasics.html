BorderContainer Basics

BorderContainer is a layout widget that can be customized to fit almost any layout. It can have up to 5 regions or panes: 4 optional panes (top, bottom, left and right) and a mandatory center pane.

How to create and configure a BorderContainer

  1. Open the Dojo Containers section of the Widgets palette.
  2. Drag a BorderContainer onto the canvas. The Border Container Dialog will pop up.
  3. Select Headline, and uncheck the Right pane.
  4. Click Ok to close the dialog window.

The BorderContainer default size is set to 100% of the window. You can change this by setting a fixed height and width in the Properties palette Layout section.

How to view or edit a BorderContainer's configuration

  1. Select the BorderContainer in the Outline palette.
  2. On the canvas, scroll the window down until you see the selection border at the very bottom of the canvas.
  3. Double-click on the selection border to bring up the Border Container Dialog.

To bring up the Border Container Dialog you must double-click on the widget's border on the canvas.
