BorderContainer Panes

BorderContainer panes are constructed using nested ContentPane widgets:

  • The top and bottom panes are specified with a height only.
  • The left and right panes are specified with a width only.
  • The center pane should have no specified dimension.

By default, BorderContainer panes can be resized by the user. You can create fixed panes by turning off the splitters.

How to set the size of BorderContainer panes

There are two ways to set the size of BorderContainer panes:

  • With the BorderContainer widget selected, drag the pane borders with your mouse to their desired locations.
  • With a single pane selected, set the height or width in the Layout section of the Properties palette.

Note: When a pane's splitter is turned off, you cannot resize the pane by dragging on its border.
