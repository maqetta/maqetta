About BorderContainer Splitters

Splitters allow the user to interactively resize the panes in a BorderContainer. They are initially visible as a small thin line inside the gutter. When you hover your mouse over a splitter, the splitter color changes to a light blue, indicating that you can now move the border by dragging it with your mouse.

Splitter behavior in Maqetta

  • In the Maqetta Page Editor:
    • Pane size changes made with the splitters is persistent.
    • Resizing a pane with the splitter will update the pane's width or height.
    • The splitters are only active when the BorderContainer widget is selected.
    • The splitters are not active when one of the panes is selected.
  • In the final page (Preview in Browser):
    • Pane size changes with the splitters is not persistent.
    • When the page is refreshed the pane sizes will return to their original positions.

How to turn off the splitters

To set a pane to a fixed size with no splitters, you can turn off the splitter as follows:

  1. Select the pane whose splitter you want to turn off.
  2. Go to the Widget Specific section of the Properties palette and uncheck Splitters.
