The Maqetta Widget palette contains GUI widgets and HTML tags that are the building blocks for mocking up web pages in Maqetta.

In this section, the term widget refers to both widgets and HTML tags. While HTML tags are technically not widgets, in Maqetta they behave in a similar fashion in that they are objects that can be dragged and dropped onto the canvas, and can be customized by setting their properties using the Styling and Object palettes. See the HTML section for a comparison of the differences between widgets and HTML tags.

Widget Libraries

Maqetta uses widgets from the following open source Javascript widget libraries:

  • Dojo Toolkit - an open source Javascript toolkit (or library) managed by the Dojo Foundation.

The widgets under Dojo Controls and Dojo Containers are from the Dojo Toolkit (or more specifically, from Dojo's UI Library Dijit).

The Widget Palette

The widgets in the Widget palette are grouped into five sections: Dojo Controls, Dojo Containers, HTML, Clip art, and Drawing Tools.

  • Dojo Controls are widgets that are typically used on input forms, such as TextBox, ComboBox, CheckBox, RadioButton and Button.
  • Dojo Containers are widgets that can contain other widgets, and are typically used for page layout; creating sections, windows or portals; or to delineate an aggregate or grouping of widgets.
  • The HTML section contains some common HTML tags such as DIV, SPAN and LABEL.
